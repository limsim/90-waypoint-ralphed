## Codebase Patterns
- Domain files live in src/domain/; all imports use .js extension (ESM, NodeNext resolution)
- Tests use node:test + node:assert/strict; run via npm test (tsc -b then node --test glob)
- After removing/renaming source files, run npm run clean before npm test to purge stale dist/ outputs
- Point and Bounds are pure value objects in src/domain/ with no DOM/Canvas deps
